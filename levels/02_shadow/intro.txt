## Уровень 2
**Задание: Парсинг таблиц**

Вам предоставлен файл shadow_fragment.db.

**Инструкция:**
1. Определите разделитель полей в первой строке заголовка — это символы между словами type и date.
2. Используя этот разделитель, найдите строку, где поле level равно operator.
3. Извлеките значение из столбца code для этой строки.

Сохраните полученное число одной строкой в файле **answer.txt** в этой папке.
