# Уровни квеста

Каждый уровень — каталог `levels/NN_имя/` (двузначный префикс задаёт порядок).

| Файл | Назначение |
|------|------------|
| `intro.txt` | Брифинг оператора (`@ROOT@`, `@PLAYER@`, `@HOST@`, `@FLAG@`) |
| `on_success.txt` | Сообщение при верном ответе |
| `on_reject.txt` | Подсказка при ошибке (`@ANS@` — ввод игрока) |
| `generate.sh` | `source` из движка: создаёт артефакты, задаёт **`LEVEL_CORRECT`** |

Примеры команд (спойлеры): [solutions.txt](solutions.txt)
