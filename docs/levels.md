# Уровни

Каждый уровень — каталог `levels/NN_имя/`:

| Файл | Назначение |
|------|------------|
| `intro.txt` | Условие задачи (обычный текст / markdown) |
| `on_success.txt` | Сообщение при верном ответе |
| `on_reject.txt` | Сообщение при ошибке; плейсхолдеры `@ans@` и `@ANS@` |
| `generate.sh` | Создание файлов задания, переменная **`LEVEL_CORRECT`** |

Подсказки по командам: [solutions.txt](solutions.txt)
