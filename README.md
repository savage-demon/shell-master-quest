# Shell Master

Квест в терминале на знание Bash: уровни с генерацией файлов и проверкой ответа.

## Структура проекта

```
├── bin/shell-master   # запуск игры
├── lib/               # движок
│   ├── run.sh         # главный цикл по уровням
│   └── common.sh      # UI, проверка ответа, утилиты
├── levels/            # сценарии (01_sector, 02_shadow, …)
├── share/             # общие ресурсы (outro.txt)
├── scripts/           # вспомогательные скрипты (сборка standalone)
└── docs/              # документация и спойлеры
```

## Запуск

Сначала показывается приветственный экран с путём к базовой папке (по умолчанию в текущем каталоге создаётся `quest`, внутри неё соседние `level_1`, `level_2`, …). Во втором терминале перейдите в нужный `level_N`, запишите ответ в **`answer.txt`**, затем в окне квеста нажмите **Enter** для проверки (между уровнями — ещё один Enter по запросу).

```bash
./bin/shell-master
./bin/shell-master другой_каталог   # база с другим именем вместо quest
```

## Сборка

- Только упакованный Bash‑артефакт (без shc):

  ```bash
  ./scripts/gen_standalone.sh
  bash ./shell-master_standalone.sh
  ```

- Бинарник через **shc** + компилятор:

  ```bash
  make help     # список целей (то же, что просто make)
  make all
  make clean    # артефакты сборки; make distclean — ещё ./quest и ./terminal_game_*
  ```
