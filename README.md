# Shell Master

Квест в терминале на знание команд.

## Запуск

Есть два способа запуска: напрямую и через Docker.

### 1. Напрямую

Запуск в одну команду из GitHub:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/savage-demon/shell-master-quest/main/shell-master_standalone.sh)
```

Локальный запуск из репозитория:

```bash
./bin/shell-master
./bin/shell-master другой_каталог   # база с другим именем вместо quest
```

Каталог для игры должен быть новым или пустым.
