# terminteractive

Тест на знание Bash: уровни с генерацией файлов и проверкой ответа в терминале.

## Структура проекта

```
├── bin/game5          # запуск квеста
├── lib/               # движок
│   ├── run.sh         # главный цикл по уровням
│   └── common.sh      # UI, проверка ответа, утилиты
├── levels/            # сценарии (01_sector, 02_shadow, …)
├── share/             # общие ресурсы (outro.txt)
├── scripts/           # вспомогательные скрипты (сборка standalone)
└── docs/              # документация и спойлеры
```

## Запуск

Сначала показывается приветственный экран с абсолютным путём к каталогу заданий — во втором терминале выполните `cd` в эту папку, затем нажмите Enter в окне квеста.

```bash
./bin/game5
./bin/game5 имя_каталога_сохранения
```

## Сборка

- Только упакованный Bash‑артефакт (без shc):

  ```bash
  ./scripts/gen_standalone.sh
  bash ./game5_standalone.sh
  ```

- Бинарник через **shc** + компилятор:

  ```bash
  make
  ```
